### YourFleetManager
Web CRUD Application designed to manage a fleet of vehicles.

## technology used :
 #- Backend :
     - Spring Boot
     - Spring Data JPA
     - JPA & Hibernate
     - Thymleaf
     - JUnit
     - Mockito
     - Lombok
 
 #- Frontend : 
     - HTML
     - Boostrap CSS
     
 # - Databases : 
     - MySQL
     - PostgreSQl
