### YourFleetManager
Web CRUD Application designed to manage a fleet of vehicles.

## technology used :
 # - Backend :
     - Spring Boot
     - Spring Data JPA
     - JPA & Hibernate
     - Thymeleaf
     - JUnit
     - Mockito
     - Lombok
 
 # - Frontend : 
     - HTML
     - Boostrap CSS
     
 # - Databases : 
     - MySQL
     - PostgreSQl
